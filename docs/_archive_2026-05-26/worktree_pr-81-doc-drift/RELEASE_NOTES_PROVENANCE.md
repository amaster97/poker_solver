# Release Notes Provenance

This file documents the chain of release-notes / release-headlines /
release-checklist drafts that were prepared during the v1.5.1 →
v1.7.x release trajectory but were never committed to `main`.

## 2026-05-23 v1.5.1 announcement drafts (pruned)

Three drafts were prepared at the workspace root for a v1.5.1
public announcement that was reframed before publication:

- `RELEASE_NOTES_2026-05-23.md` — comprehensive notes (v1.5.1)
- `RELEASE_HEADLINES_2026-05-23.md` — Twitter/HN/blog snippets
- `RELEASE_CHECKLIST_2026-05-23.md` — pre-publication checklist

These drafts framed v1.5.1 as the current public release and the
v1.4.0 `.dmg` as the latest installer. By the time a public
announcement actually shipped, the project had progressed through
v1.6.0, v1.7.0, and v1.7.2, so the drafts were superseded by the
actual GitHub release notes for `v1.7.0`. The drafts are pruned;
this file is the provenance trail.

For the actual shipped release notes, see the GitHub release pages:

- v1.7.0: <https://github.com/amaster97/poker_solver/releases/tag/v1.7.0>
- v1.7.2: (queued — re-packaged `.dmg`, post PR #42 fork-bomb fix)

For the `.dmg`-related history that supersedes anything in the
pruned drafts, see [`dmg_spawn_loop_rca_2026-05-26.md`](dmg_spawn_loop_rca_2026-05-26.md).
